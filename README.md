# HD44780 example ESP32 Espruino
Example of using HD44780 LCD 16x2 display module with ESP32 on Espruino


LCD	Function	Espruino
1 VSS	GND	GND
2 VDD	VCC	Bat (5V)
3 VO	Contrast	GND or variable resistor
4 RS	RS	A0
5 RW	RW	GND
6 E	EN	A1
7 D0	D0	-
8 D1	D1	-
9 D2	D2	-
10 D3	D3	-
11 D4	D4	C0
12 D5	D5	C1
13 D6	D6	C2
14 D7	D7	C3
15 A	LED +	Bat
16 K	LED -	GND (5V)
